# binance4j-wallet

Binance4j-wallet is a Java library providing an API for interacting with the Binance wallet endpoints.

## Installation

### Maven

```xml
<dependency>
  <groupId>com.binance4j</groupId>
  <artifactId>binance4j-wallet</artifactId>
  <version>1.0.3</version>
</dependency>
```
